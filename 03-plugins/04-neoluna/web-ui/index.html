使用方法

NeoLuna插件中运行以下脚本：

1

loadfile('3rd/neolua/webui/server.lua')()

然后在浏览器中访问http://localhost:4000/

小技巧：
在WinForm界面的“选项窗口-设置-托盘单击”中填入webui://localhost:4000，可以实现点下托盘图标，调出WebUI专用窗口。
上面的设置项还可以填http://localhost:4000，调出浏览器。或者填快捷方式(.lnk)的路径启动外部程序。

更多信息请看WebUI项目。

这个插件设计用于调用 core 的 API 命令。  
{{< figure src="../../images/plugins/plugin_commander.png" >}}

###### 注意事项
 * 关闭窗口后程序会继续执行，直到禁用插件或者退出 V2RayGCon
 * 多行“参数”会自动合并成一行
 * “环境变量”可以用逗号分隔，也可以每个一行，但变量名和值都不可以有逗号

小技巧：  
保存时在名字前添加序号，可以调整列表显示位置。

V2RayGCon手册

V2RayGCon 是 windows 下 xray-core 及 v2ray-core v4.x 的图形(graphic)配置器(configer)。

安装

先安装 .net framework 4.5+（win10已自带），然后下载解压 V2RayGCon-box.zip 到任意目录。
